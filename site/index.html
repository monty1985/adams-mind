Getting Started

Install in 3 Steps

Get up and running with Adams Mind skills in your Claude Code environment.

01

Add the marketplace

Register the Adams Mind plugin repository as a marketplace source in your Claude Code configuration.

bash 
# Add Adams Mind as a plugin marketplace
claude plugin marketplace add git+https://github.com/monty1985/adams-mind.git
02

Install a plugin

Pick any skill from the catalog and install it. Use the plugin name prefixed with the marketplace slug.

bash 
# Install the sprint planning skill
claude plugin install pm-sprint-planning@adams-mind

# Or install a dev skill
claude plugin install dev-code-review@adams-mind
03

Use the skill

Invoke the skill directly from the Claude Code prompt using its slash command. It's immediately available in any session.

claude code 
# Run in your Claude Code session
/sprint-planning

# Or try other installed skills
/code-review
/new-engineer
/roadmap-builder
