# Product Tour

## Phase 1: Overview

- Introduce the product's core mission and target users
- Explain the primary problem it solves and the value it delivers
- Provide a high-level map of the main product areas
- Share links to the staging environment and demo accounts for hands-on exploration

## Phase 2: Feature Walkthrough

- Demo each major feature area in order of user importance
- Walk through the primary user flows end-to-end (signup, core action, key outcome)
- Highlight recent launches and what problems they solved
- Explain any non-obvious UX decisions and the rationale behind them

## Phase 3: Key Metrics

- Share the north-star metric and how it is measured
- Review the current dashboard: active users, retention, conversion, revenue
- Explain how the team defines and tracks success for each feature area
- Point to internal analytics tools and how to access them
